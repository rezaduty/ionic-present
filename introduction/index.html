Ionic چیست؟

مقدمه ای بر ionic

ionic یک فریمورک جاوااسکریپت برای تولید برنامه موبایل است

Ionic یک فریم ورک برای توسعه برنامه است که با استفاده از کتابخانه محبوب AngularJS درست شده است. AngularJS یک کتابخانه MVC است که برای ساخت برنامه های تک صفحه ای برای وب و موبایل کاربرد دارد. Ionic یک محصول از Drifty است که به عنوان استارتاپ در سیلیکن ولی شروع به کار کرد.

اصلا چرا javascript و ionic

    چرا javascript ?

    • java  اگر با java آشنایی ندارید بد نیست به سراغ javascript بروید
    • سخت افزار مناسب  اگر سخت افزار مناسب ندارید بد نیست با javascript به مقصود خود برسید
    • خروجی چند گانه  با یک بار تلاش چند خروجی بگیرید
    • راحت تر ریبایی را ترسیم کنید  با سادگی هر چه تمام تر برنامه ای زیبا بسازید
    چرا ionic ?
    • جامعه آموزشی گسترده  با چند جست و جو منابع در دسترس شما است
    • فریمورک ها دیکر چی ؟فریمورک هایی دیگری در این زمینه هستند که معروف ترینش react native
    • و همین   بله نمیخواهیم جنگی درست کنیم تجربه شخصی هر کس تعیین کننده است

راه اندازی محیط توسعه



Requirement

$ Install Android Sdk
$ Install nodeJS >= 6.1
$ Install Ionic && Cordova
$ Install Genymotion(Optional)
$ vscode (Optional)

Windows

$ Nodejs Download
$ npm install -g cordova ionic
$ sudo npm cache clean -f
$ sudo npm install -g n
$ sudo n stable

Mac

$ Nodejs Download
$ xcode-select --install
$ npm install -g ios-deploy
$ npm install -g cordova ionic
$ sudo npm cache clean -f
$ sudo npm install -g n
$ sudo n stable

Linux 

$ sudo apt-get install nodejs
$ npm install -g cordova ionic
$ sudo npm cache clean -f
$ sudo npm install -g n
$ sudo n stable

ساختار برنامه

  • Tabs App
  • Blank App
  • Side menu app

ساخت یک برنامه tabs app

          
              ionic start myApp tabs
              cd myApp
              ionic cordova platform add ios 
              ionic build android

ساخت یک برنامه blank app

          
            ionic start myApp blank
            cd myApp
            ionic platform add android
            ionic cordova platform add ios 
            ionic run android

ساخت یک برنامه side menu app

          
            ionic start myApp sidemenu
            cd myApp
            ionic cordova platform add ios 
            ionic run android

چگونه برنامه خود در حالت اجرا ببینم

          
            ionic serve --lab
